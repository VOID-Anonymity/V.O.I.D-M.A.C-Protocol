**M.A.C** — это экспериментальный протокол многослойного шифрования, созданный в недрах «Эфира» для обеспечения абсолютной тишины в цифровом пространстве.

## 🛠 Алхимия Шифрования (Как это работает)

Протокол накладывает три печати на ваши данные, превращая их в нечитаемый шум:

1.  **Слой M (Morse-Hex):** Текст преобразуется в байты, затем в HEX-код, и только потом в Азбуку Морзе. Это создает первый барьер и ломает стандартную структуру языка.
2.  **Слой A (AES-256-CBC):** Данные упаковываются в «золотой стандарт» симметричного шифрования с использованием 256-битного ключа, сгенерированного через *scrypt*.
3.  **Слой C (ChaCha20):** Финальный поток данных проходит через высокоскоростной шифр ChaCha20, добавляя второй уровень криптографического хаоса.
